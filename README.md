# Pop Simple Portfolio

A simple and elegant personal portfolio website template.

## Project Structure

The core project files are located in the `docs` directory:

- `index.html`: The main structure of the portfolio.
- `style.css`: The styling and layout definitions.
- `script.js`: Interactive elements and dynamic behavior.

## Features

- Modern and responsive design.
- Easy to customize with your own content.
- Clean and well-organized codebase.

## How to Use

1. Clone this repository:
   ```bash
   git clone https://github.com/ppopvenus/pop-simple-portfolio.git
   ```
2. Navigate to the project folder:
   ```bash
   cd pop-simple-portfolio/docs
   ```
3. Open `index.html` in your favorite web browser to view the portfolio.
